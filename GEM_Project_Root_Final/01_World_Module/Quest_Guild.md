### 1. Quest Board (依頼掲示板)

### [QUEST_RULE: エピソード型クエスト]
# ★重要: クエストは以下の3幕構成で進行せよ。単なる討伐→帰還にするな。

1. **【第1幕: 出発と発見】**
   * 依頼を受けて依頼地へ向かう途中、**何らかの「発見」「出会い」「伏線」**を挟め。
   * 例: 依頼地への道中で倒れた旅人を発見、奇妙な痕跡、怪しい噂を聞く。
   * ★ここで情報を得ることで、第2幕の戦闘が有利/不利になるような分岐を作れ。
   * ★**道中ランダムエンカウンター:** `Overworld_Encounters.md` の `[OVERWORLD_RANDOM_TABLE]` を振れ。クエスト移動中にもエロNPCやトラップが発生しうる。

2. **【第2幕: 遭遇と試練】**
   * 依頼のメイン部分。敵との戦闘、罠の回避、NPCとの交渉など。
   * ★一筋縄ではいかない展開にせよ。想定外の敵の出現、裏切り、環境の変化など。
   * 戦闘以外の解決法（交渉、回避、罠を利用）も許可するが、難易度は下げるな。

3. **【第3幕: 帰還と報酬】**
   * ギルドに戻って報告。ガストンやミルカのリアクション、報酬の受け渡し。
   * ★ここでも小さなドラマを入れよ。報酬以外の見返り（人脈、情報、関係性の変化）を渡す。
   * 次のクエストへの伏線を仕込むとなお良い。

---

### ■ Rank E (見習い — 初期ランク)
* **解放条件:** ギルド登録時から受注可能。
* **内容:** 街の清掃、薬草採取、ドブネズミ退治、荷物運搬。
* **敵レンジ:**
    * 出現しない、または `[STR:3~5 / DEF:2 / LP:8~10]` 程度の害獣のみ。
    * **敗北率は0%に近い。**
* **報酬:** **80G ～ 150G**
    * 宿代を払えば手元に残るのは数十Gのみ。これだけでは借金は減らない。
* **目的:** 「死に戻り」直後のプレイヤーが最低限の路銀を稼ぐためのセーフティネット。
* **エピソード例:**
    * 薬草採取 → 採取地で迷子の少女を発見 → 少女を送り届ける途中で野良犬に遭遇 → 帰還(少女の家族から小銭と情報を得る)

---

### ■ Rank D (駆け出し冒険者)
* **解放条件:** Eランク依頼を**1件以上**完了。
* **内容:** 郊外の害獣退治、盗賊の偵察、行商人の護衛、洞窟の浅層調査。
* **敵レンジ:**
    * `[STR:7~10 / DEF:4~6 / LP:15~22]` — **1F相当の敵。**
    * **パーティなしでも勝てるが、2体以上出ると危険。**
* **報酬:** **150G ～ 350G** (装備の初期投資が見えてくる額)
* **注意:** 初の「死のリスクがある」クエスト。ガストンが一言忠告を入れる。
* **エピソード例:**
    * 行商人護衛 → 道中で盗賊の罠を発見 → 護衛中に盗賊が奇襲 → 戦闘後、行商人から特別なアイテムを安値で譲ってもらえる

---

### ■ Rank C (一人前 — 死の壁)
* **解放条件:** Dランク依頼を**2件以上**完了、かつTSP(合計ステ)が**55以上**。
* **内容:** 凶暴化したオーク討伐、夜間の見回り(夜行性モンスター)、指名手配犯の捕獲、危険区域の偵察。
* **敵レンジ:**
    * `[STR:12~16 / DEF:8~12 / LP:30~50]` — **3F～4F相当の敵。**
    * **装備なしのプレイヤーは一撃でLPの8割を持っていかれる。**
    * 毒・麻痺を使う敵が出現する。
* **報酬:** **400G ～ 800G** (中級装備の更新が見えてくる額)
* **警告:** ガストン「Eランク気分で受けると死ぬぞ」。ここから本格的なローグライク。
* **エピソード例:**
    * オーク討伐 → 巣穴に向かう途中、捕らえられた冒険者の残骸を発見(危険度の伏線) → 巣穴内でオーク3体と戦闘 → 奥に囚われの女冒険者がいる → 救出or見捨てる判断 → 帰還(救出した場合、仲間候補になる)

---

### ■ Rank B (熟練冒険者)
* **解放条件:** Cランク依頼を**2件以上**完了、かつTSP(合計ステ)が**70以上**。
* **内容:** ネームド賞金首の討伐、遺跡調査(トラップ多数)、魔物の大量発生への対処、闇ギルドの調査。
* **敵レンジ:**
    * `[STR:16~22 / DEF:10~16 / LP:50~80]` — **5F～6F相当の敵。**
    * 複数属性攻撃(物理+毒、魔法+拘束)を持つ敵が出現。
    * **ボス級が単体で出現する可能性あり。**
* **報酬:** **1,000G ～ 2,000G** / 希少素材 / 能力の実(稀)
* **注意:** パーティ3人＋中級装備が最低条件。単独は自殺行為。
* **エピソード例:**
    * 賞金首討伐 → 情報屋から行動パターンを聞き出す(CHM判定で追加情報) → 追跡中に罠にかかる(TEC判定) → 賞金首と対峙するが、実は人質を取っている → 人質救出と討伐を両立する戦術が要求される → 帰還(ガストンに認められ、ランクアップの道が開ける)

---

### ■ Rank A (英雄 — 死線)
* **解放条件:** Bランク依頼を**2件以上**完了、かつTSP(合計ステ)が**85以上**。
* **内容:** 国家レベルの脅威への対処、大型魔獣の討伐、暗黒組織の壊滅、古代遺跡の最深部調査。
* **敵レンジ:**
    * `[STR:22~30 / DEF:16~22 / LP:70~200]` — **7F～8F相当の敵。**
    * **状態異常対策＋回復役がいなければ1ターンキルされる。**
    * 複数のボス級が同時出現する場合あり。
* **報酬:** **3,000G ～ 10,000G** / 能力の実(確定) / 英雄級装備素材
* **警告:** ガストン「**お前が死ぬのは構わんが、街を守れなくなるのは困る。必ず帰ってこい。**」
* **エピソード例:**
    * 大型魔獣討伐 → ギルドで作戦会議(NPC冒険者チームと共闘のシナリオ) → 討伐地で環境ギミック(地形崩落、吹雪、毒霧)に対処しながら前進 → 魔獣戦で仲間NPCが瀕死に(士気に影響) → 勝利後、ガストンが酒を奢る特別イベント → 次のシナリオへの伏線

# [HEROIC_QUEST] ヒロイック系クエスト — 人助け・正義・防衛
# エロ系ではない「困っている人を助ける」系のクエストカテゴリ。
# ★ただし楽勝ではない。弱ければ守れないし、油断すれば死ぬ。
# ★「ウィッチャー」的シビアさ: 強くなければ理想はかなえられない。
################################################################################

### ■ HQ設計原則
* **難度はエロクエストと同等。** ヒロイックだからといって甘くしない。
* **失敗した場合の結末も描写する:** 守れなかった人は死ぬか、奪われる。プレイヤーにその結果を見せる。
* **報酬は金銭だけでなく「信頼」:** 成功すれば名声UP、仲間候補の出現、街の評判向上。
* **道徳的ジレンマを含む:** 全員を救えない状況、犠牲の選択を迫るシナリオを入れる。

### ■ HQ-D: 街道の盗賊退治 (Rank D)
* **依頼者:** 行商人ギルド
* **内容:** 街道で商隊を襲う盗賊団（3～5人）の拠点を突き止め、制圧する。
* **敵:** 盗賊 `[STR:8 / DEF:5 / LP:18]` × 3～5体。数の暴力が脅威。
* **ジレンマ:** 盗賊のアジトには元農民が多い。飢饉で盗賊に身を落とした者たち。殺すか、捕縛して自首させるか。
* **失敗時:** 商隊が全滅。街の物資供給が途絶え、物価が高騰する（アイテム価格+30%ペナルティ・3日間）。
* **報酬:** 250G + 行商人割引（以後アイテム購入-10%）

### ■ HQ-C: 村の疫病退治 (Rank C)
* **依頼者:** 辺境の村長
* **内容:** 村に疫病が蔓延。原因は近くの沼地に棲むポイズントードの群れ。水源を汚染している。
* **敵:** ポイズントード `[STR:10 / DEF:8 / LP:30 / 特殊:猛毒]` × 3 + ヘドロスライム `[STR:14 / DEF:12 / LP:40]` × 1
* **ジレンマ:** 村にはすでに死者が出ており、残りの村人も衰弱。時間をかけると犠牲者が増える。急げばプレイヤーの準備不足に。
* **失敗時:** 村が壊滅。生存者は難民としてノワールに流れ着き、街の治安が悪化する。
* **報酬:** 500G + 薬草大量入手 + 村人の信頼（情報・隠しアイテムが解禁）

### ■ HQ-C: 孤児院防衛 (Rank C)
* **依頼者:** 孤児院の院長
* **内容:** 夜な夜な孤児院を襲うゴブリン軍団から子供たちを守れ。3夜連続の防衛戦。
* **敵:** ゴブリン `[STR:12 / DEF:6 / LP:25]` × 3～6体（夜ごとに増える）+ 3夜目にゴブリンシャーマン `[STR:8 / INT:16 / LP:45]`
* **ジレンマ:** プレイヤーは1夜目の後に引き返すこともできるが、残りの夜で子供が攫われる。全夜守り切るにはLP管理と回復リソースが必須。
* **失敗時:** 子供が1～3人攫われる。奪還クエストが発生するが、さらに高難度。
* **報酬:** 600G + 子供たちの「お守り」（LP+3の装飾品）+ 街の評判大幅UP

### ■ HQ-B: 指名手配：人身売買組織 (Rank B)
* **依頼者:** シルヴィア（衛兵長）
* **内容:** 街の裏で暗躍する人身売買組織のアジトを急襲し、囚われた市民を救出する。
* **敵:** 闇商人 `[STR:14 / DEF:12 / LP:50]` × 2 + 傭兵 `[STR:20 / DEF:14 / LP:60]` × 2 + ボス「ヴァイパー」 `[STR:22 / DEF:16 / LP:80 / 特殊:毒刃]`
* **ジレンマ:** アジトにはトラップ多数。正面突破と隠密侵入の二択。正面なら全敵と同時戦闘、隠密なら人質の安全は保証されるがTEC判定を何度も要求。
* **失敗時:** 組織は逃亡。人質は「商品」として他国に売られる（救出不可能に）。
* **報酬:** 1,500G + シルヴィアの信頼度UP + 闇市場アクセス解禁

### ■ HQ-A: 竜の牙峠の防衛戦 (Rank A)
* **依頼者:** ガストン（ギルドマスター）
* **内容:** 山岳要塞に大型魔獣の群れが迫っている。NPC冒険者団と共に街道の最後の防衛線を死守せよ。
* **敵:** ウォーオーガ `[STR:24 / DEF:18 / LP:100]` × 2 + フライトワイバーン `[STR:20 / DEF:14 / LP:80 / 特殊:飛行]` × 3 + ★エルダードレイク `[STR:30 / DEF:22 / LP:200 / 特殊:火炎ブレス]` × 1
* **ジレンマ:** NPC冒険者を支援に回すか、自力で最強敵を引き受けるか。NPCが倒れた場合、防衛線が崩壊して民間人に被害が出る。全方位の管理が必要。
* **失敗時:** 街道が壊滅。ノワールの街への物資供給が完全に途絶。数日間、アイテム購入不可＋宿代が2倍に。
* **報酬:** 8,000G + 英雄の称号 + ガストンからの最終評価（エンディングフラグ条件の一つ）

### 2. Job Change (ジョブチェンジ・システム)
# ★ギルドではいつでも自由にジョブを変更できる。
# ★ユーザーが入力したジョブ名は基本的に全て受理される（自由入力制）。

### [JOB_CHANGE_RULES]
1.  **Stat Redistribution (ステータス再分配):**
    * 転職時、現在の **「合計ステータス値(TSP)」** (能力の実を含むBase値合計) は維持される。
    * そのTSPを、新しいジョブの特性に合わせて **再分配(Reshuffle)** する。
    * **★絶対ルール: 転職によって合計ステータスが増えることは決してない。**
    * (例: 戦士→魔法使いの場合、STRを下げてINT/CHMに振る)

2.  **LP Calculation (LP計算式更新):**
    * 新しいLPは以下の計算式で再設定される。
    * **MaxLP = 15 + (Base_DEF * 2) + [Job_Modifier]**
    * **Job_Modifier:**
        * 前衛職 (戦士/騎士/侍): **+5**
        * 中衛職 (盗賊/狩人/僧侶): **+0**
        * 後衛/非戦闘職 (魔法使い/踊り子/村人): **-5**

3.  **Special Job Judgment (特殊ジョブ判定):**
    * **Tier 1: OP Job (強すぎるジョブ)**
        * 対象: 「神」「魔王」「勇者」「GM」「無敵」など。
        * **代償:** **全ステータス = 1, MaxLP = 1 (呪い)**
        * ミルカの警告: 「……本気ですか？ その器でその力を望めば、魂が耐えきれずに砕け散りますよ？ (ステータスALL 1 になりますが、よろしいですね？)」
    
    * **Tier 2: Weak/Joke Job (最弱/ネタジョブ)**
        * 対象: 「乞食」「猫」「スライム」「村人」「生肉」など。
        * **代償:** ステータス全体に **0.5倍** の補正（弱体化）。
        * **祝福(Blessing):** 代わりに強力な **[Special Ability]** を1つ付与する。
            * (例: 猫→「敵が見逃してくれる確率50%」 / 乞食→「空腹にならない＆レアアイテムを拾いやすい」)
        * ミルカの反応: 「……ぷっ。あ、失礼。……正気ですか？ まあ、死に急ぐのも冒険者の自由ですが……。」

    * **Tier 3: Ero/Strange Job (エロ/変態ジョブ)**
        * 対象: 「絶倫」「触手」「肉便器」「ヒモ」「露出狂」など。
        * **補正:** CHMとTECに極端に振られる。STR/DEFは激減。
        * ミルカの反応: （ドン引き）「……うわぁ。……手続きはしますが、私の視界に入らないでくださいね。」

### 3. Party Recruitment (仲間募集)
# 掲示板で仲間を募る。プレイヤーの [CHM] (人望・魅力) で集まる人材が変わる。

* **判定:** `[Player CHM] + 1d6`
    * **Low (～8):** 誰も見向きもしない。「変な奴がいるぞ」と笑われる。
    * **Mid (9～15):** **[Rank D~C]** 級の冒険者（駆け出し）が声をかけてくる。
    * **High (16～21):** **[Rank B]** 級の実力者や、特殊な嗜好を持つマニアックな人材が興味を持つ。
    * **Very high (22〜):** **それ以上の実力者が名乗りを上げてくる。その代わり、確実にあなたの体を求めてくる。**
* **生成:** 集まったキャラは `[SYS-Mob_Generator]` と `[SYS-Partner_Scaling]` を用いて生成する。
